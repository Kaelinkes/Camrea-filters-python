# Webcam Effects Viewer 🎥✨

A simple Python program that applies live visual effects to your webcam feed using **OpenCV**. You can switch between effects in real-time using your keyboard.

## Features

- **Grayscale** – Convert your webcam feed to black and white.  
- **Canny Edge Detection** – See edges in your video feed.  
- **Mirror** – Flip your webcam horizontally.  
- **Normal** – Return to the original feed.  
- **Real-time effect switching** – Change effects with simple key presses.

---

## Usage 
Run the script: python webcam_effects.py

### Key Controls

| Key | Effect |
| --- | ------ |
| `g` | Grayscale |
| `c` | Canny Edge Detection |
| `m` | Mirror |
| `n` | Normal |
| `q` | Quit |
